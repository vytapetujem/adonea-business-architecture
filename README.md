# ADONEA Business Architecture

Tento repozitár je jediný zdroj pravdy pre filozofiu, rozhodnutia, organizačný model, procesy a požiadavky na AdoneaTeam.

Aplikácia má odrážať realitu firmy. Dokumentácia v tomto repozitári určuje, akú realitu má systém podporovať.

## Štruktúra

- `docs/aios/BOOK-0-CONSTITUTION/` – základné princípy a ADR
- `docs/aios/BOOK-1-ORGANIZATION/` – organizačný systém, katalóg a karty pozícií
- ďalšie knihy budú pribúdať podľa rozvoja systému

## Pravidlo zdroja pravdy

Rozhodnutie nie je záväzné, kým nie je zapísané v tomto repozitári a označené príslušným stavom.
