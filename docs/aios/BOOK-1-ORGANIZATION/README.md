# BOOK 1 – Organization

Cieľom tejto knihy je vytvoriť Organization Engine – systém pozícií, kariet, zodpovedností, právomocí, KPI, zastupovania a pravidiel rastu.

## Základný model

`Oblasť → Pozícia → Karta pozície → Osoba`

Jedna osoba môže dočasne držať viac kariet. Každá karta však musí byť jasne pridelená konkrétnej osobe alebo označená ako neobsadená.

## Typy kariet

- systémová karta – trvalá zodpovednosť firmy
- dočasná karta – časovo obmedzené poverenie alebo projektová zodpovednosť

## Najbližšie výstupy

1. katalóg organizačných oblastí
2. katalóg všetkých budúcich pozícií
3. štandard karty pozície
4. pravidlá priraďovania a odoberania kariet
5. pravidlá vzniku novej pozície
6. prvá karta: Obchodník – Follow-up
